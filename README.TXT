Movie Recommendation System
This repository contains a Movie Recommendation System project implemented without machine learning. The system suggests movies based on predefined logic, such as popularity, genres, or user preferences.

Project Overview
The goal of this project is to build a functional Movie Recommendation System using straightforward techniques, such as:

Popularity-based recommendations
Genre-based filtering
Personalized user preferences based on predefined rules
Features :
Popularity-Based Recommendations:
Recommends the most-watched or highly-rated movies.
Genre-Based Recommendations:
Suggests movies from specific genres based on user input.
User History-Based Recommendations:
Filters recommendations by user preferences and viewing history.
